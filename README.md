# Netflix_movie-data-analysis_project


## 📌 Project Overview

This project focuses on analyzing a movie dataset to extract meaningful insights about genres, ratings, popularity, and trends. The goal is to demonstrate data cleaning, SQL querying, and data visualization skills using real-world data.

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy)
* **SQL (MySQL)**
* **Matplotlib & Seaborn** (Data Visualization)
* Jupyter Notebook / PyCharm

---

## 📂 Project Structure

* `analysis.ipynb` → Data cleaning, preprocessing, and visualization
* `queries.sql` → SQL queries used to extract insights
* `dataset.csv` → Raw dataset used for analysis

---

## 🔍 Key Analysis Performed

### 📊 Data Cleaning & Preprocessing

* Handled missing values
* Converted data types
* Cleaned column names into snake_case
* Removed duplicates

---

### 🧠 SQL Analysis

* Total number of movies
* Average movie ratings
* Genre-wise distribution
* Top-rated and most popular movies
* Year-wise trends
* Customer-like segmentation using ratings

---

### 📈 Visualization

* Genre distribution
* Popularity trends
* Rating analysis
* Comparative charts for better insights

---

## 💡 Key Insights

* Some genres consistently have higher average ratings
* High popularity does not always guarantee high ratings
* Movie production has increased over time
* A small number of movies dominate popularity

---

## 🚀 Skills Demonstrated

* Data Cleaning & Preprocessing
* SQL Query Writing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Insight Generation & Storytelling

---

## 📎 How to Use

1. Open the notebook (`analysis.ipynb`)
2. Run all cells step-by-step
3. Execute SQL queries from `queries.sql` in MySQL
4. Explore visualizations and insights

---

## 📌 Conclusion

This project showcases the complete data analysis workflow — from raw data to actionable insights — using Python and SQL. It reflects practical, real-world analytical thinking and problem-solving skills.
